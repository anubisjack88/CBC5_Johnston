Ball Jump Max

Ball Jump Max is a cool platform game that can be played using the directional keys. The game will be over if the player collides with an object or bounces out of the bounds of the game. The player will be able to see the score and this will be displayed at the "Game Over" screen. This game is intended for players age 3 years and up in regards to ability to control the character. This game has not been verified by ESRB however, the creator follows the guidelines.
